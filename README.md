OVERVIEW

The goal of this four-week training program was to learn modern web development and build a simple chat application step by step. It began with creating a basic user interface using HTML and CSS, then adding interactivity with JavaScript. Later, the project was rebuilt using React to organize it into reusable components and improve structure. Finally, Firebase Authentication was added to enable secure Google Sign-In.
Throughout the training, key web development concepts were learned and applied, including layout design, DOM manipulation, React fundamentals, state management, asynchronous programming, and cloud-based authentication.


WEEK 1
In the first week of training, I focused on learning the basics of web development. I studied HTML and CSS through video lectures and reference materials like MDN Web Docs. I also learned important concepts such as CSS Grid, and basic Git & GitHub usage.
As part of the task, I created a simple chat application layout using HTML and CSS. The page was divided into two main sections: a left sidebar and a right chat area. The sidebar included an search bar, and a list of contacts, while the chat area contained a header, message section, and input box.
I used Flexbox to arrange the two-column layout and styled message bubbles differently for sent and received messages. I also made the layout responsive so that on mobile devices, the sidebar hides and only the chat area is visible.
Overall, this week helped me understand how to structure web pages and design responsive layouts using HTML and CSS.
