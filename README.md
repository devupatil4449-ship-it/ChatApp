OVERVIEW

The goal of this four-week training program was to learn modern web development and build a simple chat application step by step. It began with creating a basic user interface using HTML and CSS, then adding interactivity with JavaScript. Later, the project was rebuilt using React to organize it into reusable components and improve structure. Finally, Firebase Authentication was added to enable secure Google Sign-In.
Throughout the training, key web development concepts were learned and applied, including layout design, DOM manipulation, React fundamentals, state management, asynchronous programming, and cloud-based authentication.


WEEK 1
In the first week of training, I focused on learning the basics of web development. I studied HTML and CSS through video lectures and reference materials like MDN Web Docs. I also learned important concepts such as CSS Grid, and basic Git & GitHub usage. As part of the task, I created a simple chat application layout using HTML and CSS. The page was divided into two main sections: a left sidebar and a right chat area. The sidebar included an search bar, and a list of contacts, while the chat area contained a header, message section, and input box. I used Flexbox to arrange the two-column layout and styled message bubbles differently for sent and received messages. I also made the layout responsive so that on mobile devices, the sidebar hides and only the chat area is visible.
Overall, this week helped me understand how to structure web pages and design responsive layouts using HTML and CSS.


WEEK 2
In the second week of training, I learned the basics of JavaScript and how to make web pages interactive. I worked with DOM manipulation to access and modify HTML elements dynamically. For the chat application, I selected the message input field and send button using JavaScript. When the user clicked the send button or pressed the Enter key, the application read the input text and created a new message bubble dynamically. The message was then added to the chat area along with a timestamp showing the current time. I also implemented features such as clearing the input field after sending a message and automatically scrolling to the latest message. These tasks helped me understand event handling, DOM manipulation, and dynamic content creation in JavaScript.
Overall, this week strengthened my understanding of how JavaScript adds functionality and interactivity to web applications.


WEEK 3
In the third week, I learned the fundamentals of React and how to build applications using reusable components. I replaced the default content in the React project and organized the application into separate components for better structure and maintainability. For the chat application, I created a components folder and developed individual components such as Sidebar, ChatWindow, MessageBubble, and MessageInput. Each component was responsible for a specific part of the user interface. The MessageBubble component received data through props to display sent and received messages, while the MessageInput component handled message entry.
All the components were then imported and assembled in App.jsx to create the complete chat application interface. This approach helped me understand component-based architecture, props, and code organization in React.
Overall, this week improved my understanding of React and how complex user interfaces can be built using small, reusable components.
